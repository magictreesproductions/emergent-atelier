# 🛠️ Tools & Resources

## Software Reverse Engineering

### Ghidra
**NSA's Software Reverse Engineering (SRE) Framework**
- **Repository**: [NationalSecurityAgency/ghidra](https://github.com/NationalSecurityAgency/ghidra)
- **Stars**: 65.4k+
- **Description**: Free and open source reverse engineering tool
- **Latest Release**: 12.0.4
- **Use Cases**: Binary analysis, malware analysis, vulnerability research, software auditing
- **Features**:
  - Decompiler support for multiple architectures
  - Collaborative reverse engineering
  - Scripting with Python and Java
  - Extensive processor support

### Why Ghidra Matters
Ghidra democratizes reverse engineering - what was once locked behind expensive proprietary tools is now freely available to security researchers, developers, and curious minds worldwide.
